# QA Automation Portfolio

Este repositório reúne projetos de automação de testes desenvolvidos ao longo da minha jornada de aprendizado, incluindo estudos práticos, treinamentos e projetos pessoais.

O objetivo é demonstrar minhas habilidades em Quality Assurance, aplicando diferentes ferramentas, linguagens e boas práticas de automação em cenários reais e simulados.

## 🔧 Tecnologias e Ferramentas

- Testes Web: Cypress, Playwright, Selenium
- Testes de API: Postman / automação integrada
- Testes Mobile: Appium / Robot Framework
- Testes Não Funcionais: JMeter, k6
- Linguagens: Java, JavaScript, TypeScript, Python

## 🧪 Tipos de Testes

- Testes End-to-End (E2E)
- Testes de API
- Testes Mobile
- Testes de Performance (não funcionais)
- Testes exploratórios e validações funcionais

## 🎯 Objetivo do Repositório

Este portfólio tem como foco:

- Praticar e evoluir minhas habilidades em automação de testes
- Explorar diferentes ferramentas e abordagens de QA
- Consolidar conhecimentos adquiridos em cursos e experiências práticas
- Demonstrar organização de código, estrutura de testes e boas práticas

## 📁 Observações

Os projetos aqui presentes representam diferentes cenários de aprendizado e evolução técnica, podendo variar em complexidade e abordagem conforme o contexto de estudo.
